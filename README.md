# 🖥️ IT Help Desk / Cloud Admin Lab (Active Directory & Windows Server)

## Overview
This repository documents a structured hands-on IT training lab focused on building real-world skills for **Help Desk, Systems Administration, and Cloud/IT Support roles**.

The environment simulates a small enterprise domain using **Windows Server Active Directory (AD DS)** and includes practical user management, group policy fundamentals, and break/fix troubleshooting scenarios.

---

## 🎯 Goals

- Build and manage a Windows Server domain environment
- Learn Active Directory Users and Computers (ADUC)
- Practice real-world Help Desk ticket scenarios
- Develop foundational Systems Administration skills
- Prepare for entry-level IT roles (Help Desk / SysAdmin / Cloud Support)

---

## 🧱 Lab Environment

- Virtualization: VirtualBox
- OS: Windows Server (Domain Controller)
- Domain: sandoval.local
- Roles Installed:
  - Active Directory Domain Services (AD DS)
  - DNS Server
